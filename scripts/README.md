# Scripts

Utility scripts for development, testing, and deployment.

## Available Scripts

### Development
- `start-backend.bat` - Start backend server
- `start-frontend.bat` - Start frontend development server
- `start-full-system.bat` - Start both backend and frontend

### Testing
- `test-backend.js` - Backend API tests
- `test-frontend.js` - Frontend component tests
- `test-integration.js` - End-to-end integration tests

### Database
- `init-database.js` - Initialize database schema
- `populate-sample-data.js` - Add sample data for testing
- `backup-database.js` - Create database backup

### Deployment
- `build-production.bat` - Build production assets
- `deploy.js` - Deploy to production environment
